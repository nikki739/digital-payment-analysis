# 📊 Digital Payment Systems Analysis in India (RBI + UPI)

## 🔹 Overview
This project analyzes digital payment trends in India using RBI and UPI datasets. It compares different payment systems such as UPI, NEFT, RTGS, IMPS, and AEPS to understand growth patterns, transaction volume, and adoption trends.

---

## 🔹 Tech Stack
- Python (Pandas, NumPy)
- Matplotlib
- Data Analysis

---

## 🔹 Key Features
- Data cleaning and preprocessing  
- Time-series analysis  
- Growth rate calculation  
- Market share analysis  
- Comparative analysis (UPI vs NEFT vs RTGS vs IMPS)  
- Rolling average and trend smoothing  

---

## 🔹 Datasets
- RBI Daily Digital Payments Data  
- UPI Monthly Transactions Data  

---

## 🔹 Notebook Files
- `rbi_payment_analysis.ipynb` → RBI payment system analysis  
- `upi_transaction_analysis.ipynb` → UPI transaction trends  

---

## 🔹 Key Insights
- UPI dominates digital payment volume  
- Traditional systems show slower growth  
- Digital payments adoption is increasing rapidly  
- Higher transaction activity observed on weekdays
